# Python Calculator

A modern desktop calculator built with **Python** and **Tkinter**, featuring a clean graphical interface and support for essential arithmetic operations. This project demonstrates the fundamentals of GUI development, event handling, and application logic in Python.

---

## Features

* Basic arithmetic operations

  * Addition (+)
  * Subtraction (-)
  * Multiplication (×)
  * Division (÷)
* Percentage (%)
* Positive / Negative toggle (+/-)
* Decimal number support
* All Clear (AC) button
* Automatic removal of unnecessary decimal zeros
* Clean and responsive Tkinter interface
* Automatically centered application window

## How It Works

The calculator follows a simple operation flow:

1. Enter the first number.
2. Select an arithmetic operator.
3. Enter the second number.
4. Press **=** to calculate the result.

The application internally stores:

* **A** → First operand
* **B** → Second operand
* **operator** → Selected arithmetic operator

Once the calculation is completed, the state is reset for the next operation.
